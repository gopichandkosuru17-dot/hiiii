## Step 1: Create GitHub Account
1. Go to https://github.com
2. Sign up for a free account if you don't have one

## Step 2: Create New Repository
1. Click "New" button (green button)
2. Repository name: `fake-account-detector`
3. Make it **Public**
4. Don't initialize with README (we'll upload our files)
5. Click "Create repository"

## Step 3: Upload Your Files
1. Click "uploading an existing file"
2. Drag and drop ALL these files:
   - frontend/index.html
   - frontend/styles.css
   - frontend/script.js
   - backend/api.py
   - backend/requirements.txt
   - backend/dataset_generator.py
   - README.md

## Step 4: Enable GitHub Pages
1. Go to your repository
2. Click "Settings" tab
3. Scroll down to "Pages" in left sidebar
4. Source: "Deploy from a branch"
5. Branch: "main"
6. Folder: "/ (root)"
7. Click "Save"

## Step 5: Get Your Public Link
After 2-3 minutes, you'll get a link like:
```
https://yourusername.github.io/fake-account-detector/frontend/
```

**Everyone in the world can access this link!**

## Step 6: Share Your Link
Send that link to anyone - they can access your website from anywhere!

Note: The ML backend won't work on GitHub Pages (it's frontend only), but the interface will be beautiful and people can see your work!
